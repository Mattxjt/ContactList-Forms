# ContactList-Forms
ContactList helps you add contacts and save it to a text file.

Features: 
 * Add contacts
 * Delete contacts
 * Save to a .txt file
 * Image Size: small or large
 
 Required contact info: 
  - First Name
  - Last Name
  - Phone Number
  - Email
